Vesion
Prerequisites
Before running the project, make sure you have the following installed on your system:

Git - Required to download the project from GitHub

Download from: git-scm.com


Node.js - Required to run and manage project packages

Download from: nodejs.org




Note: Both Git and Node.js must be installed before proceeding with the installation.

Installation and Setup
After installing Git and Node.js, follow these steps:
Run the Installation Script
Simply run the Vesion.bat file. The script will automatically:

Install pnpm package manager globally
Clone the project from GitHub
Navigate to the project folder
Install all required dependencies
Build the project
Inject the extension into your browser

Wait for Completion
You'll see a "Press any key to continue" message when the installation is finished.
